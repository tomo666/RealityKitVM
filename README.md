# Jelly Bean Game Engine

This is a Jelly Bean Game Engine (JBGE) implementation for Unity / Godot / RealityKit powered by JB Script VM.
Dev Environment: MacBook Pro M1 Max - Mac OS Tahoe 26.2 / Xcode 26.2

Requirements:
- Unity: Unity 6000.3.3f1 / VSCode
- Godot: Godot Standard 4.4.1, SwiftGodotKit
- RealityKit: --
